Что важно по дизайну:

- красно-черная айдентика
- крупная типографика
- много воздуха
- аккуратные hover-эффекты
- мягкие reveal-анимации при скролле
- дорогие карточки, секции, кнопки
- можно использовать glass / blur / subtle glow, но очень умеренно
- нельзя делать дешевые неоновые эффекты
- нельзя превращать сайт в “киберпанк”
- дизайн должен быть эстетичным и взрослым

Что важно по детской странице:

По UX:

- фиксированный header
- красивое меню
- заметные CTA-кнопки
- удобная мобильная версия
- продуманные отступы и визуальная иерархия

По анимациям:
- использовать Framer Motion
- анимации должны быть премиальными и плавными
- не делать навязчивых и слишком частых анимаций
- нужны:

  - reveal on scroll для секций
  - hover animation для карточек и кнопок
  - плавные transitions между блоками
- если где-то уместно, предложи parallax или layered motion, но без перегруза
